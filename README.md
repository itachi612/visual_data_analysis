# Data Visualization (Pandas, Matplotlib, Seaborn, Plotly)

## О проекте

Проект посвящён визуализации и анализу данных с использованием Python.  
Основная цель — научиться представлять данные в наглядном виде и извлекать из них полезные инсайты.

В рамках проекта используются реальные данные, которые обрабатываются с помощью SQL и Pandas, а затем визуализируются с помощью различных библиотек.

---

## Что реализовано

В проекте реализованы различные типы визуализаций:

- линейные графики (line chart)
- столбчатые диаграммы (bar chart)
- гистограммы (histogram)
- boxplot
- scatter matrix
- тепловые карты (heatmap)
- интерактивные графики (Plotly)

---

## Используемые технологии

- Python  
- Pandas  
- SQLite  
- Matplotlib  
- Seaborn  
- Plotly

### Вывод

Проект помог закрепить навыки визуализации данных и понять, как графики используются для анализа и принятия решений.

Визуализация позволяет не только представить данные, но и глубже понять их структуру и закономерности, что является важной частью работы с данными.
